# In Progress

仍在开发中的 skills。它们还没准备好发布；预期会有 rough edges、breaking changes 和被放弃的实验。毕业到稳定 bucket 前，它们会从 plugin 和顶层 README 中排除。

- **[review](./review/SKILL.md)** — 从固定点开始按两个轴线 review 变更：**Standards**（diff 是否符合 repo coding standards）和 **Spec**（diff 是否忠实实现来源 issue/PRD）。
- **[teach](./teach/SKILL.md)** — 在当前 workspace 中围绕一个 topic 维护 mission、glossary、resources 和 learning records，进行多 session 教学。
- **[writing-beats](./writing-beats/SKILL.md)** — 将文章塑造成 beat journey，类似 choose-your-own-adventure。选择 starting beat，只写这一 beat，然后 pivot 到下一 beat，直到文章自然结束。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 通过 grilling session 挖出 fragments，也就是异质的写作 nuggets，并追加到单一文档，作为未来文章的 raw material。
- **[writing-shape](./writing-shape/SKILL.md)** — 接收一份 raw material markdown 文件，并逐段塑造成文章，在每一步讨论格式选择。
