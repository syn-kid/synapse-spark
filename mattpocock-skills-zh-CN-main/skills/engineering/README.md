# Engineering Skills

日常代码工作使用的 skills。

- [diagnose](./diagnose/SKILL.md) — 面向困难 bug 和性能回退的纪律化诊断循环。
- [grill-with-docs](./grill-with-docs/SKILL.md) — 对照现有 domain model 和文档化决策，对计划进行 grilling 并内联更新 `CONTEXT.md` / ADR。
- [improve-codebase-architecture](./improve-codebase-architecture/SKILL.md) — 发现 codebase 中的 deepening 机会。
- [setup-matt-pocock-skills](./setup-matt-pocock-skills/SKILL.md) — 为 engineering skills 搭建 issue tracker、triage labels 和 domain docs 的每仓库配置。
- [tdd](./tdd/SKILL.md) — 使用 red-green-refactor 循环做 test-driven development。
- [to-issues](./to-issues/SKILL.md) — 将计划、spec 或 PRD 拆成 issue tracker issues。
- [to-prd](./to-prd/SKILL.md) — 将当前上下文整理成 PRD 并发布到项目 issue tracker。
- [triage](./triage/SKILL.md) — 通过 triage roles state machine 分诊 bugs 和 feature requests，并准备 agent briefs。
- [zoom-out](./zoom-out/SKILL.md) — 请求更高层次的代码上下文。
- [prototype](./prototype/SKILL.md) — 构建 throwaway prototype 来细化 logic/state 或 UI 设计。
