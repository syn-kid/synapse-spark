## 1. 整体架构

双层结构：无限画布 + 底部固定控制栏。色彩与质感全面更新为"暗室星图"系统。

## 3. 画布区域

### 3.1 背景与空状态

| 属性 | 值 |
|------|-----|
| 背景 | 虚空黑 `#0D1117`（绝非纯黑） |
| 初始状态 | 单一闪烁光标，颜色星光 `#E6EDF3`（不透明度 0.6） |
| 引导文字 | 无 |

### 3.2 节点：三色系统

身份差异不再靠颜色体现，而是靠节点内部的微型标签或上下文识别。

| 颜色 | 名称 | 语义 | 使用场景 |
|------|------|------|----------|
| `#7C3AED` | 突触紫 | 激活 / 操作 / 当前选中 | 当前被选中的节点（无论已访问/未访问）。用户的注意力锚点。 |
| `#3B82F6` | 树突蓝 | 足迹 / 已访问历史 | 用户曾经探索过的路径节点（已访问但非当前选中）。走过的路。 |
| `#DB2777` | 轴突粉 | 收藏 / 灵感标记 | 用户手动收藏的节点（独立于访问状态）。想要保留的念想。 |
| `#484F58` | 暗星云 | 未访问 / 沉睡态 | 未走过的分支节点。视觉后退，不争夺注意力。 |

### 3.3 节点形状与状态编码（双重编码规则）

- 所有节点：全圆角 pill（圆角半径 ≥18px），无尖角

| 状态 | 样式 |
|------|------|
| **未访问** | 描边空心 — 背景透明，边框 1.5px 暗星云 `#484F58`，文字星尘 `#8B949E` |
| **已访问** | 实心填充 — 背景树突蓝柔光 `rgba(59,130,246,0.10)`，边框无或极淡，文字星光 `#E6EDF3` |
| **当前选中** | 实心填充 — 背景突触紫柔光 `rgba(124,58,237,0.15)`，边框 1.5px 突触紫 `#7C3AED`，文字星光 `#E6EDF3`，放大 115%，字重 700 |
| **收藏** | 在已访问或未访问基础上，右上角添加轴突粉色小圆点标记（直径 6px），不改变主体颜色 |

### 3.4 尺寸与字体

| 属性 | 值 |
|------|-----|
| 默认高度 | 36px，padding 左右 16px |
| 未访问分支 | 85% 缩放 |
| 选中节点 | 115% 缩放 |
| 字体 | Inter（全界面唯一字体） |
| 衬线体 | 无装饰衬线体出现在节点中（规范明确：衬线体仅限 logo） |

### 3.5 连线

| 类型 | 样式 |
|------|------|
| 已走路 | 实线，1.5px，树突蓝 `#3B82F6`（不透明度 0.25） |
| 未走路 | 虚线，1px，暗星云 `#484F58` |
| 箭头 | 无箭头，无端点装饰 |
| 弧度 | 有机弧度，不笔直 |
| 排斥力 | 防重叠 |

### 3.6 模型 Logo 小图标

| 属性 | 值 |
|------|-----|
| 位置 | 节点左上角 |
| 尺寸 | 8×8px（克制） |
| 颜色 | 星尘 `#8B949E`（不让模型标识抢走三色系统的注意力） |

## 4. 动效与响应

### 4.1 默认动效

所有过渡使用 `cubic-bezier(0.16, 1, 0.3, 1)`（符合规范缓出曲线）。

| 动效 | 时长 |
|------|------|
| 关联词辐射出现 | 280ms |
| 选中放大 | 150ms |
| 面板弹出 | 200ms |
| 页面入场 | 无编排式动画（规范：产品工具直接进入任务） |

### 4.2 prefers-reduced-motion

全部即时切换，时长 0ms。

### 4.3 响应目标

每次交互 < 50ms 提供视觉反馈（规范要求）。

|------|------|

| 焦点环 | 2px 实线，突触紫 `#7C3AED`，对比度 ≥ 4.5:1 |
| 触控目标 | ≥ 44px |

## 6. 对照旧版修正清单

| 修正项 | 原方案 | 修订后 |
|--------|--------|--------|
| 节点色彩逻辑 | 按身份分色（琥珀/青灰/钢蓝等） | 按状态分色（紫选中/蓝已访/粉收藏/灰沉睡） |
| 背景色 | 笼统深灰 `#1C1C1E` | 虚空黑 `#0D1117`（非纯黑） |
| 节点字体 | 衬线体 | Inter（统一字体规则） |
| 阴影 | 未明确禁用 | 完全取消投影，用背景色层级抬升 |
| 动效曲线 | 多个不同曲线 | 统一为 `cubic-bezier(0.16, 1, 0.3, 1)` |
| 圆角 | 部分 12px | 统一 pill 或 ≥24px |
| 颜色禁用项 | — | 确认无 `#000`、无渐变文字、无 `border-left` 色条 |
